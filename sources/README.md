# Sources

This folder stores project-specific reference materials for a future writing task.

Recommended contents:
- source papers
- notes from meetings or interviews
- outlines
- tables or figures exported from analysis
- equations or derivation notes
- policy, product, or domain reference documents

Recommended practice:
- keep original source files here when possible
- use descriptive file names
- do not store long-term project truth here; freeze that content into `shared/` after confirmation
