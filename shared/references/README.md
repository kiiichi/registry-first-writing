# Shared References

This directory stores example entries and format references for the active files under `shared/`.

Use this directory only when:
- clarifying file shape or schema,
- checking example granularity,
- bootstrapping a new project registry,
- or refactoring the template itself.

Do not treat files in `shared/references/` as active project truth.
Do not load them by default during drafting, review, or pre-submission.
